When can this app help me?

  • The touchscreen has broken on your mobile device, or the display is broken.
  • You need to remotely view notifications and input messages on your mobile device.
  • Work on your own mobile application and the need to have visual control and interactive interaction with it.
  • Will help to make a presentation of the mobile application on the big screen.
  • Manage your mobile device using the keyboard and mouse from a PC or laptop.

Mobile screen on Desktop

Launch applications, play games, debug your own programs, manage your mobile device on your desktop! Use the keyboard and mouse to navigate, typing from the keyboard is much more efficient!

Features:

  • built-in terminal for access to the console commands of the remote device
  • support for LUA scripts to interact with the device and applications on it
  • built-in graphics editor / LUA script generator
  • The system of loadable plugins in the assembled package includes a plug-in for managing LUA scripts
  • supports configuration file with basic interface settings
  • the mode of displaying absolute coordinates both in pixels and positions in the video buffer
  • save screenshots in .bmp or .png formats or copy to clipboard
  • a set of typical Android commands for quick access from the menu
  • install APK packs

Bender Rodriguez edition

In the unconnected state, Bender - a famous character in the Futurama series, will be able to share with you his thoughts about the universe, do not miss his coming!

Built-in terminal:

  • supports console terminal mode on a remote device
  • editable text entry field, copy and paste text, command history, character input control
  • adjustable indent from the bottom edge of the screen when opening the terminal
  • maintains a history of entered commands and a list of predefined shortcut commands

Interactive Interaction

Interactive interaction with mobile applications is very easy to implement using LUA scripts. The built-in script editor allows you to create a script visually, knowledge of the LUA scripting language is not required. The scripts you create will automatically execute the script after the plugin starts.

Compatibility and system requirements:

  • uses ADB to access the device
  • image acquisition and processing is done through our own drivers temporarily copied to the device, uses AScreencap as driver
  • supports devices with Android versions from 5.x to 9.x and Android API from version 20
  • currently available ready-made assembly for Windows 10 32bit in the format of the msi archive (Bender Rodriguez edition)
