Основные возможности:

  • встроенный терминал для доступа к консольным командам удаленного устройства
  • поддержка скриптов LUA для интерактивного взаимодействия с устройством и приложениями на нем
  • встроенный графический редактор/генератор скриптов LUA
  • система подгружаемых плагинов в собранный пакет включен плагин управления LUA скриптами
  • поддерживает конфигурационный файл с основными настройками интерфейса
  • режим отображения абсолютных координат как в пикселях, так и позиции в видео буфере
  • сохранение скриншотов в форматах .bmp или .png или копирование в буфер обмена
  • набор типовых команд Android для быстрого вызова из меню
  • установка APK пакетов

Встроенный терминал:

  • поддерживает режим консольного терминала на удаленном устройстве
  • редактируемое поле ввода текста, поодерживает копирование и вставку текста, историю команд, контроль ввода символов
  • настраеваемый отступ от нижнего края экрана при открытии терминала
  • поддерживает историю введенных команд и список заранее заданных команд быстрого доступа

Интерактивное взаимодействие

Интерактивное взаимодействие с мобильными приложениями очень просто реализовать с помощью LUA скриптов. Встроенный редактор скриптов позволяет создавать скрипт визуально, знания скриптового языка LUA не требуется. Созданные вами скрипты атоматический выполнят сценарий после запуска плагина.

Совместимость и системные требования:

  • использует ADB для доступа к устройству
  • получение и обработка изображения производиться через собственные драйвера временно копируемые на устройство
  • в качестве драйвера использует AScreencap
  • поддерживает устройства с версиями Android от 5.x до 9.x и Android API от 20 версии
  • на текущий момент доступна готовая сборка под Windows 10 32bit в формате архива msi
